# Notes Organizer 

A simple Notes Organizer web application built using **HTML, CSS, and JavaScript**.  
It allows users to create, edit, delete, and manage personal notes easily.

## 🚀 Features

-  Add new notes
-  Edit existing notes
-  Delete notes
-  Save notes using localStorage
-  Simple and responsive UI

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
